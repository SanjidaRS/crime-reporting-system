# Crime Reporting System

## Overview

A cloud-based crime reporting system that allows citizens to securely report crimes online.

This project follows a microservice architecture and is being developed as a learning project to understand:

- Git & GitHub
- Backend Development
- Frontend Development
- PostgreSQL & Supabase
- Docker
- CI/CD with GitHub Actions
- Microsoft Azure
- Kubernetes

---

## Planned Features

- Submit crime reports
- Upload evidence
- Track report status
- Admin dashboard
- Crime analytics

---
